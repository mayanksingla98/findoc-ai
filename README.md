# FinDoc AI

A production-grade financial document intelligence API built with Node.js, 
OpenAI, PGVector, and LangGraph.

Upload any financial document — bank statements, annual reports, invoices — 
and get accurate, grounded answers via a hybrid RAG pipeline. Ask complex 
questions through an agent that routes across 4 tools. Parse bank statements 
into structured, categorised transactions with anomaly detection and 
explainable flags.

Built to demonstrate what separates a real AI engineering system from a 
ChatGPT wrapper — cost control, observability, structured output reliability, 
PII compliance and production failure handling.
